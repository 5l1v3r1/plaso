# Small INSTALL file.

Mac OS X:

Prepare:
Install XCode, since we are about to start compiling bunch of stuff.

Install these libraries:
+ install FUSE (http://osxfuse.github.com/)
+ iPython (http://archive.ipython.org/release/0.13/ipython-0.13.tar.gz)
+ pytz (http://pypi.python.org/pypi/pytz/)
+ libvshadow (remember to use ./configure --enable-python) and remember to read the README.macosx
+ libregf (remember to use ./configure --enable-python)$ and remember to read the README.macosx
+ pytsk3 (http://code.google.com/p/pytsk/wiki/Installing)
    (requires you to set up mercurial if not done so already)

Actual Work:
1. Get this repo, check.
2. Download protobuf: http://code.google.com/p/protobuf/downloads/list (http://protobuf.googlecode.com/files/protobuf-2.4.1.tar.bz2)
    (standard ./configure, make and make install should do)
3. Compile the proto
    (run ./compile_proto.sh)
